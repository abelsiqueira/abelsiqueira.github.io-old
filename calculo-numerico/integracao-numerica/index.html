Integração Numérica

Em cálculo I aprendemos que

∫abf(x)dx=F(b)−F(a), \int_a^b f(x)\text{d}x = F(b) - F(a), ∫ab​f(x)dx=F(b)−F(a),

se fff for uma função integrável, e onde FFF é uma primitiva de fff.

No entanto, nem sempre é fácil encontrar uma primitiva para FFF. Às vezes, pode ser impossível, como em

∫abe−x2dx. \int_a^b e^{-x^2}\text{d}x. ∫ab​e−x2dx.

Por sorte, muitas vezes uma aproximação numérica é suficiente para nosso objetivo.

Vamos estudar alguns métodos de aproximação numérica, a maior parte deles usando valores da função fff em pontos selecionados, isto é

∫abf(x)dx≈∑i=1mAif(xi), \int_a^bf(x)\text{d}x \approx \sum_{i=1}^m A_if(x_i), ∫ab​f(x)dx≈i=1∑m​Ai​f(xi​),

para uma quantidade de pontos mmm, pontos xix_ixi​ e valores AiA_iAi​.

Salvo indicação contrária, iremos supor que x1<x2<⋯<xmx_1 < x_2 < \dots < x_mx1​<x2​<⋯<xm​.

Fórmulas de Newton-Cotes

As fórmulas de Newton-Cotes provém da interpolação polinomial da função fff em pontos igualmente espaços do intervalo [a,b][a,b][a,b]. Se os intervalos estiverem entre os pontos escolhidos, isto é, se x1=ax_1 = ax1​=a e xm=bx_m = bxm​=b, então a fórmula é dita fechada. Caso contrário, aberta.

Uma interpolação polinomial da função fff nos pontos x1,…,xmx_1, \dots, x_mx1​,…,xm​, é uma função polinomial ppp que satisfaz p(xi)=f(xi)p(x_i) = f(x_i)p(xi​)=f(xi​), para i=1,…,mi = 1,\dots,mi=1,…,m.

As fórmulas de Newton-Cotes fechadas terão os pontos xix_ixi​ descritos por

xi=a+(i−1)h, x_i = a + (i-1)h, xi​=a+(i−1)h,

onde h=b−am−1h = \dfrac{b-a}{m-1}h=m−1b−a​.

No caso das fórmula abertas, o espaçamento entre os pontos é hhh, mas começamos a contar depois de aaa e paramos antes de bbb. Desse modo, podemos pensar que o espaçamento é hhh no intervalo fechado [a+h,b−h][a + h, b - h][a+h,b−h]. De modo que

h=b−h−(a+h)m−1. h = \frac{b - h - (a + h)}{m - 1}. h=m−1b−h−(a+h)​.

Resolvendo essa equação para encontrar hhh, obtemos

h=b−am+1. h = \frac{b - a}{m+1}. h=m+1b−a​.

Ponto-Médio

Vamos começar considerando uma simples aproximação para fff: uma interpolação de um ponto.

p(x)=f(a+b2), p(x) = f\bigg(\frac{a+b}{2}\bigg), p(x)=f(2a+b​),

isto é, ppp é uma função constante. Essa aproximação resulta em

∫abf(x)dx≈∫abf((a+b)/2)dx=(b−a)f(a+b2). \int_a^b f(x)\text{d}x \approx \int_a^bf((a+b)/2)\text{d}x = (b-a)f\bigg(\frac{a+b}{2}\bigg). ∫ab​f(x)dx≈∫ab​f((a+b)/2)dx=(b−a)f(2a+b​). 
f(x) = exp(x)
a, b = 0, 1
Iexata = MathConstants.e - 1
IPM = (b - a) * f( (a + b) / 2 )
ErroPM = Iexata - IPM
ErroRelPM = abs(ErroPM) / Iexata

function integral_PM(f, a, b)
    x = (a + b) / 2
    h = (b - a) / 2
    return 2h * f(x)
end

integral_PM(exp, 0, 1)
1.6487212707001282

Essa aproximação é chamada de regra do ponto-médio, e é uma fórmula aberta. Note que não é possível uma fórmula fechada de um ponto, por motivos óbvios.

Note que, como [a,b][a,b][a,b] foi divido em dois, temos h=(b−a)/2h = (b-a)/2h=(b−a)/2. Daí, podemos escrever

∫abf(x)dx≈2hf(x1). \int_a^b f(x)\text{d}x \approx 2h f(x_1). ∫ab​f(x)dx≈2hf(x1​).

Trapézio

A fórmula de trapézio é uma fórmula de dois pontos. Note que dado dois pontos distintos, temos uma reta. Pensando na integral como a área abaixo dessa reta, vemos que a área será a área de um trapézio. Daí, o nome.

Para o caso fechado, os dois pontos no intervalo [a,b][a,b][a,b] serão x1=ax_1 = ax1​=a e x2=bx_2 = bx2​=b.

Para o caso aberto, os dois pontos no intervalo [a,b][a,b][a,b] serão x=2a+b3x = \frac{2a + b}{3}x=32a+b​

f(x) = exp(x) + cos(3π * x) * 0.5 + 3
F(x) = exp(x) + sin(3π * x) / 6π + 3x
a, b = -2, 2
plot(f, a, b, c=:red, fill=(0,:red,0.5), leg=false)
plot!(x->f(a) + (x - a) * (f(b) - f(a)) / (b - a), a, b, c=:blue, fill=(0,:blue,0.5))
scatter!([a, b], [f(a), f(b)], c=:blue)
png(joinpath(@OUTPUT, "fig-int3"))
plot(f, a, b, c=:red, fill=(0,:red,0.5), leg=false)
x1, x2 = (2a+b)/3, (a+2b)/3
plot!(x->f(x1) + (x-x1)*(f(x2)-f(x1))/(x2-x1), a, b, c=:blue, fill=(0,:blue,0.5))
scatter!([x1, x2], [f(x1), f(x2)], c=:blue)

A reta que usamos para aproximar a integral é dada por

p(x)=f(x1)+(x−x1)f(x2)−f(x1)x2−x1, p(x) = f(x_1) + (x-x_1)\frac{f(x_2)-f(x_1)}{x_2-x_1}, p(x)=f(x1​)+(x−x1​)x2​−x1​f(x2​)−f(x1​)​,

onde x1=ax_1 = ax1​=a e x2=bx_2 = bx2​=b no caso fechado, ou x1=2a+b3x_1 = \dfrac{2a+b}{3}x1​=32a+b​ e x2=a+2b3x_2 = \dfrac{a + 2b}{3}x2​=3a+2b​.

Em ambos os casos, teremos

∫abf(x)dx≈b−a2[f(x1)+f(x2)]. \int_a^b f(x)\text{d}x \approx \frac{b-a}{2}[f(x_1) + f(x_2)]. ∫ab​f(x)dx≈2b−a​[f(x1​)+f(x2​)].

No entanto, além de x1x_1x1​ e x2x_2x2​ serem diferentes, temos também o valor de hhh. Para o caso fechado, temos h=b−ah = b-ah=b−a. Daí,

∫abf(x)dx≈h2[f(x1)+f(x2)]. \int_a^b f(x)\text{d}x \approx \frac{h}{2}[f(x_1) + f(x_2)]. ∫ab​f(x)dx≈2h​[f(x1​)+f(x2​)].

Para o caso aberto, temos h=b−a3h = \frac{b-a}{3}h=3b−a​. Logo,

∫abf(x)dx≈3h2[f(x1)+f(x2)]. \int_a^b f(x)\text{d}x \approx \frac{3h}{2}[f(x_1) + f(x_2)]. ∫ab​f(x)dx≈23h​[f(x1​)+f(x2​)]. 
function trapezio_fechado(f, a, b)
    return (b-a)*(f(a)+f(b))/2
end

function trapezio_aberto(f, a, b)
    h = (b-a)/3
    return (b-a)*(f(a+h)+f(b-h))/2
end

F(x) = exp(x)

Erro

Uma pergunta importante aqui é quão errada é essa aproximação. Note que, diferente do método de Newton ou da bissecção, não estamos calculando aproximações sucessivas para a integral, e sim uma única aproximação, que esperamos ser boa o suficiente.

O cálculo da fórmula do erro envolve outros conceitos não estudados nesta disciplina. Iremos apenas apresentar os erros, começando pelas três aproximações que vimos.

Abaixo, ξ\xiξ é algum número em [a,b][a,b][a,b] ou (a,b)(a,b)(a,b), se a função for fechada ou aberta, respectivamente.

  • Ponto médio

∫abf(x)dx=2hf(x1)+h33f′′(ξ)=(b−a)f(b−a2)+(b−a)324f′′(ξ); \int_a^b f(x)\text{d}x = 2hf(x_1) + \frac{h^3}{3}f''(\xi) = (b-a) f\bigg(\frac{b-a}{2}\bigg) + \frac{(b-a)^3}{24}f''(\xi); ∫ab​f(x)dx=2hf(x1​)+3h3​f′′(ξ)=(b−a)f(2b−a​)+24(b−a)3​f′′(ξ);

  • Trapézio fechado

∫abf(x)dx=h2[f(x1)+f(x2)]−h312f′′(ξ)=b−a2[f(a)+f(b)]−(b−a)312f′′(ξ); \int_a^b f(x)\text{d}x = \frac{h}{2}[f(x_1) + f(x_2)] - \frac{h^3}{12}f''(\xi) = \frac{b-a}{2}[f(a) + f(b)] - \frac{(b-a)^3}{12}f''(\xi); ∫ab​f(x)dx=2h​[f(x1​)+f(x2​)]−12h3​f′′(ξ)=2b−a​[f(a)+f(b)]−12(b−a)3​f′′(ξ);

  • Trapézio aberto

∫abf(x)dx=3h2[f(x1)+f(x2)]+3h34f′′(ξ)=b−a2[f(2a+b3)+f(a+2b3)]−(b−a)336f′′(ξ). \int_a^b f(x)\text{d}x = \frac{3h}{2}[f(x_1) + f(x_2)] + \frac{3h^3}{4}f''(\xi) = \frac{b-a}{2}[f\bigg(\frac{2a+b}{3}\bigg) + f\bigg(\frac{a+2b}{3}\bigg)] - \frac{(b-a)^3}{36}f''(\xi). ∫ab​f(x)dx=23h​[f(x1​)+f(x2​)]+43h3​f′′(ξ)=2b−a​[f(32a+b​)+f(3a+2b​)]−36(b−a)3​f′′(ξ).

As três aproximações compartilham uma mesma propriedade. O erro é da ordem de h3f′′(ξ)h^3 f''(\xi)h3f′′(ξ). Isso quer dizer que para valores pequenos de hhh, o erro diminui cubicamente. Note que o ξ\xiξ para cada aproximação pode ser diferente.

f(x) = exp(x)
F(x) = exp(x)
cores = [:blue, :red, :magenta]

b = [1.0; 1e-6]

Duas conclusões importantes podem ser tiradas daqui. Primeiro, não existem bons motivos para se usar o trapézio aberto, uma vez que o erro do ponto médio é da mesma ordem que o do trapézio aberto. Segundo, que todas essas fórmulas tem uma erro proporcional à f′′(ξ)f''(\xi)f′′(ξ), de modo que integram perfeitamente funções afins.

f(x) = 2x + 5
F(x) = x^2 + 5x
a, b = 0, 1
ponto_medio(f, a, b) - (F(b) - F(a))
0.0

Outro ponto agora é se conseguimos melhorar nossa aproximação num intervalo fixo [a,b][a,b][a,b]. Existem duas possibilidades: aumentar a quantidade de pontos da interpolação, ou quebrar o intervalo [a,b][a,b][a,b] em vários intervalos menores.

Regra de Simpson

Usando 3 pontos chegamos nas seguintes fórmulas:

Note que h=b−a2h = \dfrac{b-a}{2}h=2b−a​ na fórmula fechada e h=b−a4h = \dfrac{b-a}{4}h=4b−a​ na fórmula aberta.

A fórmula fechada de 3 pontos é chamado de regra de Simpson.

function simpson(f, a, b)
    h = (b - a) / 2
    return h * (f(a) + 4 * f(a + h) + f(b)) / 3
end

function aberta3pontos(f, a, b)
    h = (b - a) / 4
    return 4h * (2 * f(a + h) + 2 * f(b - h) - f(a + 2h)) / 3
end

As fórmulas com 3 pontos, como esperávamos, são melhores que as de 2 ou 1 pontos. O erro também passou a ser proporcional a f(4)(ξ)f^{(4)}(\xi)f(4)(ξ), de modo que podemos calcular a integral exata de polinômios até grau 3.

f(x) = x^3 + x^2 + x + 1

a, b = 0, 1

4.440892098500626e-16

No entanto, quando passamos para 4 pontos, não temos a mesma sorte. Assim como a transição do ponto-médio ao trapézio aberto, não temos melhoria na ordem de convergência da aproximação, nem para a fórmula fechada nem para a fórmula aberta. Esse fenômeno se repete, de modo que criar interpolações com cada vez mais pontos se torna tanto inviável quanto ineficaz.

Composição de aproximações (ou repetição)

A outra estratégia para melhorar a aproximação no cálculo da integral é quebrando o intervalo em pedaços menores e aproximando cada pedaço.

Em outras palavras, transformamos o intervalo em a=x1<x2<⋯<xm=ba = x_1 < x_2 < \dots < x_m = ba=x1​<x2​<⋯<xm​=b, com xi+1−xi=h=b−am−1x_{i+1} - x_i = h = \frac{b-a}{m-1}xi+1​−xi​=h=m−1b−a​ no caso fechado e em a=x0<x1<x2<⋯<xm<xm+1=ba = x_0 < x_1 < x_2 < \dots < x_m < x_{m+1} = ba=x0​<x1​<x2​<⋯<xm​<xm+1​=b, com xi+1−xi=h=b−am+1x_{i+1} - x_i = h = \frac{b-a}{m+1}xi+1​−xi​=h=m+1b−a​ no caso aberto.

Note que no caso aberto criamos dois pontos extras x0=ax_0 = ax0​=a e xm+1=bx_{m+1} = bxm+1​=b, para deixar mais simples o entendimento.

Com os intervalos separados, utilizamos as fórmulas dos métodos "encaixando" os pontos da fórmula na discretização que fizemos.

O exemplo mais fácil de entender é o da regra do trapézio. Veja abaixo, onde quebramos o intervalo [−2,2][-2,2][−2,2] em 4 intervalos (i.e., m=5m = 5m=5).

f(x) = exp(x) + cos(3π*x) * 0.5 + 3
a, b = -2, 2
p = plot(f, a, b, c=:red, lw=2, fill=(0,:red,0.3), leg=false)

for i = 1:m-1
    xi = a + (i-1) * h
    xip = xi + h
    L(x) = f(xi) + (x-xi) * (f(xip) - f(xi)) / h
    plot!(p, L, xi, xip, c=:blue, fill=(0,:blue,0.4))
    scatter!([xi; xip], [f(xi); f(xip)], c=:blue, ms=5)

xticks!(a:h:b, ["x$i" for i = 1:m])

Na teoria, temos agora

function trapezio_composto(f, a, b; m = 10) # Note que não faremos mais o trapézio aberto.
    h = (b - a) / (m - 1)
    I = 0.0
    xi = a
    for i = 2:m-1
        xi += h
        I += f(xi)
    end
    I = f(a) + 2I + f(b)
    return I * h / 2
end

function simpson_composto(f, a, b; n = 100) # n eh o numero de pontos
    #Implemente
    if n % 2 == 0
        n = n + 1
    end
    h = (b - a) / (n - 1)
    I2 = I4 = 0.0
    x = a + 2h
    for i = 3:2:n-2
        I2 += f(x)
        x += h
        I4 += f(x)
        x += h
    end
    return (f(a) + 4 * f(a + h) + 4I4 + 2I2 + f(b)) * h / 3
end

Quadratura Gaussiana

A ideia da quadratura Gaussiana é obter a propriedade de integrar exatamente polinômios do maior grau possível.

∫−11g(t)dt≈∑i=1mAig(ti). \int_{-1}^1 g(t) \text{d}t \approx \sum_{i=1}^m A_i g(t_i). ∫−11​g(t)dt≈i=1∑m​Ai​g(ti​).

  • g(t)=1g(t) = 1g(t)=1 implica em ∫−11dt=2=A1\int_{-1}^1\text{d}t = 2 = A_1∫−11​dt=2=A1​

  • g(t)=tg(t) = tg(t)=t implica em ∫−11dt=0=A1t1\int_{-1}^1\text{d}t = 0 = A_1 t_1∫−11​dt=0=A1​t1​
