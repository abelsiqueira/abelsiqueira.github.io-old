O problema de ajuste de um conjunto de dados à um modelo, simplesmente dito ajuste de curvas, é um problema bastante prático e recorrente. Esse problema acontece quando temos muito mais informação que o necessário para se determinar exatamente um modelo. Mas isso também só faz sentido se essa informação tiver algum tipo de erro experimental ou intrínseco.

Considere o seguinte problema de ajustar um conjunto de pontos por uma reta.

Não existe uma reta que passa por todos esses pontos. Inclusive, alguns deles podem ter o mesmo xxx, de modo que não existe nenhum função que passa por eles. Sendo assim, precisamos encontrar uma reta que aproxima estes pontos.

Essa reta é definida por L(x)=α+βxL(x) = \alpha + \beta xL(x)=α+βx, e precisamos encontrar esses valores para α\alphaα e β\betaβ.

Uma estratégia é considerar o erro de aproximação e tentar minimizá-lo. Vamos separar o erro termo a termo, obtendo assim um Erroi_ii​, que depende dos valores de α\alphaα e β\betaβ.

Nosso problema pode ser escrito como

min⁡E(α,β)=∑i=1mErroi(α,β). \min E(\alpha,\beta) = \sum_{i=1}^m \text{Erro}_i(\alpha,\beta). minE(α,β)=i=1∑m​Erroi​(α,β).

Já sabemos que para minimizar uma função, devemos buscar seus pontos críticos. Isto é,

∂E∂α=0, \frac{\partial E}{\partial \alpha} = 0, ∂α∂E​=0,

e

∂E∂β=0. \frac{\partial E}{\partial \beta} = 0. ∂β∂E​=0.

Para calcular essa derivada, devemos calcular a derivada de Erroi_ii​. Então, a escolha dessa função de erro deve ser bem pensada, para que seja significativa e fácil de derivar.

A escolha mais comum é Erroi=12[α+βxi−yi]2=12[L(xi)−yi]2_i = \frac{1}{2}[\alpha + \beta x_i - y_i]^2 = \frac{1}{2}[L(x_i) - y_i]^2i​=21​[α+βxi​−yi​]2=21​[L(xi​)−yi​]2.

Essa escolha é boa pois a derivada dessa função é fácil de calcular, e é fácil de visualizar que é o quadrado da distância vertical da reta ao ponto (xi,yi)(x_i,y_i)(xi​,yi​).

e

∂βE(α,β)=∑i=1mxi(α+βxi−yi). \partial_{\beta} E(\alpha,\beta) = \sum_{i=1}^m x_i (\alpha + \beta x_i - y_i). ∂β​E(α,β)=i=1∑m​xi​(α+βxi​−yi​).

"""`regressao_linear(x, y)`
Calcule y ~ α + β x
"""
function regressao_linear(x, y)
    m = length(x)
    A = [ones(m)  x]
    sol = (A' * A) \ (A' * y)
    α, β = sol[1], sol[2]
    return α, β
end

"""
y ~ β₀ + β₁ x + … + βₚ xᵖ
"""
function regressao_polinomial(x, y, p)
    m = length(x)
    A = ones(m, p + 1)
    for i = 1:m
        for j = 2:p+1
            A[i,j] = x[i]^(j - 1)
        end
    end
    β = (A' * A) \ (A' * y)
    return β
end

Regressão Polinomial - Implementação

"""
y ~ β₀ + β₁ x + … + βₚ xᵖ
"""
function regressao_polinomial(x, y, p)
    m = length(x)
    A = ones(m, p + 1)
    for i = 1:m
        for j = 2:p+1
            A[i,j] = x[i]^(j - 1)
        end
    end
    β = (A' * A) \ (A' * y)
    return β
end

Vamos voltar para o problema inicial de ajustar dados por uma reta: (xi,yi),i=1,…,m(x_i, y_i), i = 1,\dots,m(xi​,yi​),i=1,…,m. Vamos chamar a solução encontrada de y^=ax+b\hat{y} = a x + by^​=ax+b, isto é y^i=axi+b≈yi\hat{y}_i = a x_i + b \approx y_iy^​i​=axi​+b≈yi​.

Forma matricial

Com os vetores ϕ‾1,…,ϕ‾\overline{\phi}_1, \dots, \overline{\phi}ϕ​1​,…,ϕ​, podemos montar a matriz

A=[ϕ‾1⋯ϕ‾n]. A = [\overline{\phi}_1 \cdots \overline{\phi}_n]. A=[ϕ​1​⋯ϕ​n​].

Com essa matriz, podemos ver que ATA=M A^TA = MATA=M e que ATy=cA^Ty = cATy=c. Ou seja, o sistema que queremos resolver é, na verdade

ATAα=ATb, A^TA\alpha = A^Tb, ATAα=ATb,

ou seja, AT(Aα−b)=0A^T(A\alpha - b) = 0AT(Aα−b)=0.

Isso quer dizer que o sistema normal tem alguma relação com o sistema sobre-determinado Aα=bA\alpha = bAα=b. De fato, se pensarmos novamente no problema de minimizar aquela E(α)E(\alpha)E(α), cada termo dentro do somatório é uma linha de Aα−bA\alpha - bAα−b. Em outras palavras,

E(α)=12∑i=1m(b−Aα)i2=12∥Aα−b∥2. E(\alpha) = \frac{1}{2}\sum_{i=1}^m (b - A\alpha)_i^2 = \frac{1}{2}\Vert A\alpha - b\Vert^2. E(α)=21​i=1∑m​(b−Aα)i2​=21​∥Aα−b∥2.

De fato, se calculamos ∇E(α)=0\nabla E(\alpha) = 0∇E(α)=0, obtemos as equações normais. Isso generaliza o problema de quadrados mínimos. Esse problema aparece com frequência em outros contextos, e é bastante importante sabermos resolvê-lo eficientemente. Infelizmente, nesta disciplina, não teremos tempo de aprender como fazer isso de maneira eficiente.

Álgebra linear do problema de quadrados mínimos

Como vimos, estamos minimizando ∥Ax−b∥\Vert Ax -b \Vert∥Ax−b∥. Isso pode ser visto como min⁡∥y−x∥\min \Vert y -x\Vertmin∥y−x∥ para y∈Im(A)y \in Im(A)y∈Im(A). Ou seja, estamos calculando a projeção de bbb na imagem de AAA.

Se b=y+zb = y + zb=y+z, com y∈Im(A)y \in Im(A)y∈Im(A), então z⊥Im(A)z \perp Im(A)z⊥Im(A). Mas, Im(A)⊥=Nu(AT)Im(A)^{\perp} = Nu(A^T)Im(A)⊥=Nu(AT), de modo que z∈Nu(AT)z \in Nu(A^T)z∈Nu(AT). Em outras palavras, ATz=0A^Tz = 0ATz=0. Como y∈Im(A)y \in Im(A)y∈Im(A), então existe uuu tal que y=Auy = Auy=Au. Daí, temos b=Au+zb = Au + zb=Au+z com ATz=0A^Tz = 0ATz=0. Sendo assim,

∥Ax−b∥2=∥Ax−Au−z∥2=∥A(x−u)−z∥2=∥A(x−u)∥2+∥z∥2,\Vert Ax - b\Vert^2 = \Vert Ax - Au - z\Vert^2 = \Vert A(x - u) - z\Vert^2 = \Vert A(x - u)\Vert^2 + \Vert z\Vert^2, ∥Ax−b∥2=∥Ax−Au−z∥2=∥A(x−u)−z∥2=∥A(x−u)∥2+∥z∥2,

onde a última igualdade segue de ortogonalidade. Portanto, a escolha que minimiza ∥Ax−b∥\Vert Ax - b\Vert∥Ax−b∥, é x=ux = ux=u, já que ∥z∥\Vert z\Vert∥z∥ não pode ser modificado pela escolha de xxx. Nisso, temos

r=b−Ax=Au+z−Au=z,r = b - Ax = Au + z - Au = z,r=b−Ax=Au+z−Au=z,

isto é, rrr é a projeção ortogonal de bbb no núcleo de ATA^TAT.
